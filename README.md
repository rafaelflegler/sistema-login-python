# Sistema de Login e Pagamento em Python

Mini sistema desenvolvido em Python utilizando terminal.

## Funcionalidades

- Cadastro de usuários
- Login com email e senha
- Menu pós-login
- Sistema de saldo individual
- Realização de pagamentos
- Validação de idade
- Validação de email único

## Tecnologias

- Python

## Objetivo

Projeto criado para praticar lógica de programação, funções, dicionários e estruturas de repetição.
